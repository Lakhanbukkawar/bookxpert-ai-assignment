# Bookxpert AI Assignment

This repository contains solutions for the Bookxpert AI Assignment.

## Tasks

### Task 1: Get Matching Person Names
A fuzzy name-matching system that identifies the most similar person names from a dataset using string similarity techniques.

### Task 2: Local LLM Integration & Recipe Chatbot
A lightweight, dataset-grounded recipe chatbot built using an open-source LLM and exposed via a FastAPI backend.

Each task is implemented with clarity, robustness, and suitability for low-resource environments.
